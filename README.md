> Connect farmers with restaurants. Source local, eat local.  
> Chat with Glean to source and sell.


### The problem

Farmers struggle to list surplus or imperfect produce quickly, and restaurants spend time hunting for reliable local suppliers.

### What we do

Cultivate is a marketplace that connects farmers and restaurants through **Glean**, an AI assistant that helps farmers turn harvests into listings and helps restaurants find and source local produce in natural language.

### Local development

**Install dependencies** (from repo root):

```bash
npm install
```

**Run full stack** (frontend + backend):

```bash
npm run dev
```

Then open the frontend (e.g. `http://localhost:5173` for Vite) and ensure the app is configured to talk to the local API (e.g. `http://localhost:3000`). See `pkgs/app` and `pkgs/server` READMEs for package-specific setup and env vars.
